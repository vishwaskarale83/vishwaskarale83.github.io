    <article class="card project-card">

        <li>Clear module organization, documentation, and build automation</li>

    <article class="card project-card">
      <h3>Ethernet Hub</h3>
      <p>
        Implemented a full-duplex Ethernet hub in Verilog with frame parsing, CRC validation, MAC-based forwarding, broadcast handling, and simulation support.
      </p>
      <ul>
        <li>Modular Verilog architecture</li>
        <li>Protocol-aware frame processing and forwarding</li>
        <li>Testbench-driven validation workflow</li>
      </ul>
    </article>

    <article class="card project-card">
      <h3>RISC-V Pipeline Simulation Workbench</h3>
      <p>
        Built a five-stage RISC-V simulator covering 500+ instructions, robust instruction decoding, and performance analysis using CPI and IPC metrics.
      </p>
      <ul>
        <li>90% fewer decoding errors</li>
        <li>100+ branching and jump scenarios handled</li>
        <li>20% pipeline efficiency improvement</li>
      </ul>
    </article>

    <article class="card project-card">
      <h3>Embedded Gesture Recognition System</h3>
      <p>
        Designed a real-time gesture-based unlock system with C++, Mbed OS, gyroscope pipelines, and matching algorithms such as DTW and correlation.
      </p>
      <ul>
        <li>Modular embedded software architecture</li>
        <li>Multi-threaded coordination across sensors, input, and display</li>
        <li>Real-time user feedback on-device</li>
      </ul>
    </article>

    <article class="card project-card">
